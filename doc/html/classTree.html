Detailed Description

A class to represent a tree data structure.

The tree is implemented as a set of nodes, where each node contains information about its parent, child, sibling, and subtree size. The class provides various methods for manipulating and querying the tree.

◆ compute_supports()

py::list Tree::compute_supports ( int  u) const

Finds supports of wavelets associated to an interior node.

Note that the first returned vector contains a list of the indices of all nodes in the support of all wavelets associated with the target node. The second vector contains the boundary indices of a partition into each support.

For example, the the ith wavelet associated with 'u' takes on its left-value on leaves partition[i] to partition[i+1].

Parameters
uTarget node index.
Returns
A list of vectors, each containing the leaf-indices of the supports of the wavelets associated with node 'u'
