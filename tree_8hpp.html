Detailed Description

Header file for the Tree class and related operations.

This file defines the TreeNode structure and the Tree class, which represents a tree data structure supporting various operations such as linking, cutting, swapping subtrees, and finding paths, ancestors, and children. It also provides functions for managing edge lengths and subtree sizes, and for printing the tree structure.

The Tree class is used to model hierarchical relationships, where each node can have a parent, children, and siblings. The tree is implemented as a static structure, with a fixed number of nodes.

Functions are provided to manipulate the tree structure, including linking nodes, cutting edges, and swapping subtrees. Additionally, utility functions for traversing the tree, finding leaves, and computing the effective path length (EPL) are included.
